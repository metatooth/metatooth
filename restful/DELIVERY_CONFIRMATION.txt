RUNNING THE API
================================================================================

1. Start Docker:
   docker compose up -d

2. Install dependencies:
   bundle install

3. Start server:
   DATABASE_URL="postgresql://metatooth:metatooth@localhost:5432/metatooth_development" \
   bundle exec foreman run shotgun

4. Run tests:
   bundle exec rspec spec/

5. Access Swagger UI:
   http://localhost:9393/swagger
